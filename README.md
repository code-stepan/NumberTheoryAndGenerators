## Описание алгоритмов

### Функция Эйлера φ(n)

Функция Эйлера φ(n) — количество целых чисел от 1 до n, взаимно простых с n. Реализованы три независимых способа вычисления:

| Метод | Описание |
|-------|----------|
| `TotientByDefinition(n)` | Наивный: перебор чисел 1..n, подсчёт тех, у которых `gcd(k, n) = 1` |
| `TotientByFactorization(n)` | Формула: `n · ∏(1 − 1/p)` по всем различным простым делителям n |
| `TotientByDFT(n)` | Тригонометрический: `round(∑ gcd(k,n) · cos(2πk/n))` |

### Тест простоты Ферма

Вероятностный тест, основанный на малой теореме Ферма: если a^(n−1) ≢ 1 (mod n), то n — составное. При k случайных основаниях вероятность ошибки не превышает 2^(−k).

- `PowMod(base, exp, mod)` — быстрое модулярное возведение в степень (метод повторного возведения в квадрат).
- `FermatTest(n, k)` — тест с k случайными основаниями из [2, n−2].

> **Примечание:** тест не является надёжным для чисел Кармайкла.

### Символы Лежандра и Якоби

- **Символ Лежандра** `(a/p)` для нечётного простого p: вычисляется через критерий Эйлера `a^((p−1)/2) mod p`.
- **Символ Якоби** `(a/n)` для нечётного n: обобщение символа Лежандра с использованием законов квадратичной взаимности и правил для деления на 2.

### Тест Соловея — Штрассена

Вероятностный тест простоты. Для случайного a проверяется условие:

```
a^((n−1)/2) ≡ Jacobi(a, n) (mod n)
```

Если равенство не выполняется — n составное. Число раундов k задаёт точность.

### Тест Миллера — Рабина

Наиболее известный вероятностный тест простоты. Представление n−1 = 2^s · d. Для каждого свидетеля a проверяется:

1. a^d ≢ 1 (mod n) и a^(2^r · d) ≢ −1 (mod n) для всех r ∈ [0, s−1].

Если ни одно условие не выполнено — n, вероятно, простое. Тест корректно определяет составность чисел Кармайкла.

### Линейный конгруэнтный генератор (LCG)

Классический ПСЧ-генератор:

```
x_{n+1} = (a · x_n + b) mod m
```

- Определяет период генератора путём отслеживания повторяющихся состояний.
- `EstimateLCGVariance(samples)` — оценка дисперсии периодов при случайных параметрах a, b, m.

### Обратный конгруэнтный генератор (ICG)

Модификация LCG с использованием модулярного обратного:

```
x_{n+1} = (a · x_n^{-1} + b) mod n
```

Работает как для простых, так и для составных модулей. Включает расширенный алгоритм Евклида для вычисления обратного элемента.

### Генератор Блюма — Блюма — Шуба (BBS)

Квадратичный генератор:

```
x_{n+1} = x_n^2 mod M,   где M = p · q,   p ≡ q ≡ 3 (mod 4)
```

- `FindPrime3Mod4(max)` — поиск простого числа-множителя Блюма (p % 4 == 3).
- Период определяется свойствами M и начального значения seed.
- Дискретная логарифмическая сложность делает BBS криптографически стойким при большом M.
